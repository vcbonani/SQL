# SQL

Repositório com exercícios, consultas e exemplos desenvolvidos durante os cursos de SQL da plataforma Alura, utilizando diferentes tecnologias de banco de dados:

- SQLite
- MySQL
- Google BigQuery

O objetivo deste repositório é centralizar práticas de consultas SQL, análise de dados, procedures, funções, joins, views, transações e manipulação de dados em ambientes distintos.
